# Canvas Interface Demo

这是一个最小的 framework-only demo，用来证明：

- `canvas-interface` 可以独立承载 `CanvasUI`
- 可以独立承载 `WindowManager` / `SurfaceWindow`
- 可以独立承载 Builder surface
- 可以独立承载 Developer 基础面板

## 运行

在仓库根目录执行：

```bash
bun packages/canvas-interface/demo/index.html
```

或使用根脚本：

```bash
bun run dev:canvas-interface-demo
```
